# Slack GIF Creator

Emoji GIFs: 128x128, FPS 10-30, 48-128 colors. Uses PIL for drawing, GIFBuilder for assembly, easing functions for smooth motion. Supports shake, pulse, bounce, spin, fade, slide, zoom, and particle effects.
