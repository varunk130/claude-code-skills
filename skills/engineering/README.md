Skills for engineers and engineering leaders: API design, deployment, code explanation, issue resolution, frontend development, and code review workflows.

## Who these are for

Software engineers, tech leads, engineering managers, and anyone shipping code that needs to be reviewed, tested, deployed, or explained.

## Design principles

- **Tests before commits** — every change has a verification path
- **Consistent conventions** — names, error shapes, patterns all standardized
- **Explain like you mean it** — code explanations include the *why*, not just the *what*
- **Accessibility and responsiveness as defaults** — not afterthoughts
- **Visual when it helps** — diagrams, dashboards, and trees over walls of text
