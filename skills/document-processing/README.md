# Document Processing Skills

Skills for working with documents: extraction, transformation, merging, and form handling.

| Skill | What it produces |
|-------|------------------|
| [pdf-processing](pdf-processing/SKILL.md) | PDF text extraction, form filling, and document merging |

## Who these are for

Anyone working with PDFs, forms, and document pipelines - operations teams, legal, compliance, finance, and product teams handling document workflows.

## Design principles

- **Round-trip safe** - read, modify, and write without breaking structure
- **Form-field aware** - preserve interactive PDF features
- **Batch-friendly** - designed for many-document workflows
- **Encoding-robust** - handles non-Latin text, encrypted PDFs, scanned docs
