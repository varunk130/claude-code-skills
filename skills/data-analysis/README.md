# Data Analysis Skills

Skills for data scientists, analysts, and researchers: SQL conventions, SaaS metrics, structured data analysis, and rigorous topic research.

| Skill | What it produces |
|-------|------------------|
| [data-scientist-analysis](data-scientist-analysis/SKILL.md) | SQL best practices, SaaS metrics definitions, and structured data analysis conventions |
| [deep-research](deep-research/SKILL.md) | Thorough topic research using codebase search and file analysis tools |

## Who these are for

Data scientists, analysts, BI engineers, growth and product analysts, and researchers running rigorous quantitative or qualitative investigations.

## Design principles

- **Define metrics before measuring** — same name, same definition, every time
- **Show the query** — analysis is reproducible
- **Confidence levels, not certainty** — small samples and limitations called out
- **Bias-aware** — sample, selection, and survivorship biases flagged
- **Citation-anchored** — research output traces back to sources
