# Strategic Management Skills

Classical and modern strategy frameworks executed with rigor: industry structure, competitive positioning, landscape mapping, scenario planning, and OKR-based execution.

| Skill | What it produces |
|-------|------------------|
| [blue-ocean-strategy-canvas](blue-ocean-strategy-canvas/SKILL.md) | As-is and to-be value curves, ERRC grid, non-customer analysis, buyer utility map, sequence test |
| [go-to-market-strategy](go-to-market-strategy/SKILL.md) | Complete GTM plan: channels, messaging matrix, launch timeline, success metrics |
| [gtm-engineering](gtm-engineering/SKILL.md) | GTM data infrastructure: lead scoring, routing, lifecycle automation, attribution, reverse ETL contracts |
| [market-sizing](market-sizing/SKILL.md) | TAM / SAM / SOM via top-down and bottom-up with assumption tracking |
| [okr-cascade-planner](okr-cascade-planner/SKILL.md) | Company → function → team OKR cascade with alignment map and quarterly rituals |
| [porters-five-forces-analyzer](porters-five-forces-analyzer/SKILL.md) | Five Forces scorecard with dominant force, trajectory, and strategic moves per force |
| [pricing-strategy-analyzer](pricing-strategy-analyzer/SKILL.md) | Pricing analysis with competitive positioning, packaging, and revenue modeling |
| [swot-tows-analyzer](swot-tows-analyzer/SKILL.md) | Evidence-anchored SWOT + TOWS matrix producing prioritized strategic options |
| [wardley-mapping](wardley-mapping/SKILL.md) | User-need value chain on the evolution axis with climatic patterns, doctrine, and gameplay |

## Who these are for

CEOs, strategy leads, founders, GMs, product leaders, and corporate strategists planning multi-quarter or multi-year direction.

## Design principles

- **Structure over speculation** — frameworks make the reasoning visible
- **Pair internal with external** — capabilities and trends analyzed together
- **Multi-horizon** — today's positioning + 3-year trajectory
- **Action over description** — every analysis ends in moves with owners
- **Defendable under pushback** — every claim cites evidence
