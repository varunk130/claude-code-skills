# Financial Analysis Skills

Rigorous, defensible financial modeling and analysis skills for valuation, capital allocation, SaaS economics, cash management, and fundamental analysis.

| Skill | What it produces |
|-------|------------------|
| [ai-agent-financial-analyst](ai-agent-financial-analyst/SKILL.md) | SaaS financial models: unit economics, ROI calculators, pricing scenarios, revenue projections |
| [burn-rate-runway-planner](burn-rate-runway-planner/SKILL.md) | Burn decomposition, scenario plans, lever stack, 13-week cash forecast, fundraise trigger calendar |
| [capital-allocation-framework](capital-allocation-framework/SKILL.md) | Ranked investment portfolio across NPV, IRR, PI, EAA + strategic option value and risk-adjusted frontier |
| [dcf-valuation-builder](dcf-valuation-builder/SKILL.md) | Full DCF with WACC build, two terminal-value methods reconciled, sensitivity table, football field |
| [financial-statement-analyzer](financial-statement-analyzer/SKILL.md) | 10-K / 10-Q teardown with margin waterfall, quality-of-earnings score, accounting red-flag log |
| [saas-cohort-analyzer](saas-cohort-analyzer/SKILL.md) | Cohort retention triangles, NRR/GRR, LTV, CAC payback, movement bridge, segmentation cuts |

## Who these are for

CFOs, FP&A leads, investors, founders, strategy teams, and product leaders making financial decisions that need to hold up in an investment committee, board meeting, or audit.

## Design principles

- **Show the math** — every formula and assumption is visible
- **Cite the source** — every input ties back to data or benchmark
- **Quantify ranges, not points** — sensitivity is part of the answer
- **Surface red flags early** — accounting, working capital, and earnings-quality checks built in
- **Defensible under pushback** — outputs survive an IC or board grilling
